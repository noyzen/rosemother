Settings

Enable Logging

Records application activity for troubleshooting. Can be disabled for minor performance gains.

Prevent System Sleep

Prevents the computer from entering sleep or idle mode while a backup job is running.

Shutdown on Completion

Automatically shut down the computer after a "Start All" batch run is completed. This is not saved and must be enabled each session.

Enable Auto Cleanup

If enabled, automatically delete files from the destination if they are removed from the source. This applies to all jobs and runs after a successful backup.

Export Jobs

Save all your current job configurations to a JSON file. This file can be used for backup or to import your jobs on another machine.

Import Jobs

Load job configurations from a previously exported JSON file. Jobs that already exist (same source and destination) will be skipped.
